| CommitMessage | Mean |
| -- | -- |
| [Feat] | 기능 추가 |

## 제작 후 피드백
* 나뭇가지생성을 재귀함수로 구현해서 그려질 때 버벅임이 있다.
* 다크모드에서 색깔이 자연스럽게 바귀는 것을 제대로 구현하지 못한 것 같다.
* 구현할 때 .tsx로 확장자를 지었으나 타입스크립트 활용을 제대로 못 하였다.
* 굳이 리액트로? 구현했어야 하나 싶다.
* 프로젝트 초기 세팅에 애를 먹었디.(타입스크립트,Eslint 등등..)
